# Meow House

ร้านขายน้องแมวสายพันธุ์แท้ — Next.js 16 (App Router). รายละเอียดเพิ่มเติมที่ `README.md`

## Agent skills

### Issue tracker

Issues are tracked in **GitHub Issues** via the `gh` CLI. External PRs are **not** a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical triage labels, using their default names: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
